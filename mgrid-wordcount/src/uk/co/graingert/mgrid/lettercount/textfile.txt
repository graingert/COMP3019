run a job on an allocation created using omii_client tender using the named application. once the job has been started its status is monitored until it is finished and the results are available. it is safe to stop monitoring a job at any time use the monitor command to resume monitoring later. The input files naturally must have been previously uploaded with omii_client upload. the outputs can then be downloaded with omii_client download. work.xml is a file describing the processing requirements of this job.
